The ultimate lorem ipsum text generator for WordPress. No need for browsing dummy content, just use ALL THE ISPUMS!!!

== Description ==

Tired of searching for lorem ipsum text every time you want to toss in some dummy text while developing a site or showing concepts to a client? ALL THE IPSUMS generates lorem ipsum with 7 easy shortcodes (with room to grow!).

Pick from the following 7 shortcodes to get started with your placeholder text. 

Shortcodes:

*   [zombie]
*   [bacon]
*   [skater]
*   [pommie]
*   [batman]
*	[metaphor]
*	[kitty]

More ipsums to come! If you've got an ipsum that you must have, let me know and I'll add it in. 

== Installation ==

1. Upload `all-the-ispums` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use one of the above shortcodes in a post, page or widget
1. Sit back and marvel at how much more fun your content is
